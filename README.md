## ğŸ’¡ Project Overview

This project develops an **induction cooker** that integrates:
- **Solar Power System**: Harnesses renewable energy for sustainable operation
- **IoT Integration**: Smart monitoring and control via Battery Management System (BMS) in addition to a mobile/web application 
- **Induction Heating Technology**: Efficient, safe, and fast cooking

### Key Features
- Solar panel integration for off-grid capability
- Real-time energy monitoring through IoT sensors
- Battery management system for energy storage optimization
- Remote monitoring and control capabilities
- Energy-efficient induction heating
- Safety features and auto-shutoff mechanisms
